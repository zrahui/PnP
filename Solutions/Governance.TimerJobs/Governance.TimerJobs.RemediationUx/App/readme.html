Readme

Governance.TimerJobs.RemediationUx provides a set of UX widgets for site collection administartors to resolve non-compliances.

The front end is implemented in Angualr-JS.

The server side is powered by ASP.NET Web API.

Unlock Site Collection

http://{host}/App/index.html#/unlock/?SiteUrl={siteUrl}

The query string key SiteUrl is fixed. Its value will be filled by SPO tenant.

One should call UnlockController.SetNoAccessRedirectUrl() method before any site collection got locked, this will instruct SPO tenant redirects any further locked site accessing request to the URL above.

Edit Site Information

http://{host}/App/index.html#/siteedit/{siteUrl}

Site collection provisioning UX App should inject a site action menu item by using User Custom Action, which should be linked to this Edit Site Information page for site collection administrator to update site metadata or decommission the site.
